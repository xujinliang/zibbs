# 梓论坛 —— PHP+Bootstrap轻论坛系统

#### 为什么开放源代码？
开发者已经转型成前端程序员，年龄35+，迷茫的未来，迷茫的结局，在编程历史上留下自己的痕迹，让未来的程序员将源代码传承下去！

#### 什么是梓论坛？
Zibbs是一款基于Bootstrap和其他优秀开源插件搭建的开源的PHP轻论坛，从0开始的，性质是原创。
